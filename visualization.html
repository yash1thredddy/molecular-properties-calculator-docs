THE ANALYSES

Visualization gallery

Every chart the app can draw, when to use each, and what to read from it.

What the Data Visualization mode is for

Once you have a table of molecules and their properties — calculated in Batch Processing or uploaded as your own spreadsheet — the Data Visualization workspace turns those columns into interactive charts so you can see the patterns instead of squinting at rows of numbers.

A column is simply one named measurement that appears for every molecule — for example Molecular Weight, LogP (greasiness), or your own measured activity. You choose which column goes on which axis, and the app draws the chart live in your browser. The charts are built with Plotly (an interactive charting library), so every plot can be hovered, zoomed, panned, and saved as an image — no chart is a dead picture.

A quick vocabulary

An axis is one edge of the chart (the horizontal X axis and the vertical Y axis). A data point (or "mark") is one molecule drawn as a dot, bar, or line vertex. A distribution is the shape you get when you ask "how are the values spread out?" — are most compounds clustered around one value, or split into separate piles? Every chart below answers a slightly different question about your data.

The chart gallery

Nine chart types are available from the chart-type menu. Each card below shows a representative sketch of what the chart looks like, what it tells you, and the situation it was built for. The sketches use illustrative data, not real measurements.

Scatter plot

One dot per molecule, placed by its X and Y values. An optional trendline (the dashed line) summarizes the overall direction, and the app prints the line's equation and its R² — a 0-to-1 score for how tightly the dots hug the line (1 = perfect, 0 = no relationship).

Use when: you want to see whether two properties move together — e.g. does activity rise as weight rises?

Line chart

Connects the points in order with a continuous line, emphasizing the trajectory — the up-and-down path — rather than a cloud of separate dots.

Use when: your X axis has a natural sequence (a synthesis series, a concentration ramp, a time course) and you want to follow the trend along it.

Bar chart

One bar per category; bar height encodes a value. Great for head-to-head comparison across labelled groups (scaffolds, series, vendors).

Use when: your X axis is a label (not a number) and you want to compare a single value across those labels at a glance.

Histogram

Slices one column into value ranges ("bins") and stacks the molecules in each bin into a bar — revealing the distribution shape (one hump? two? skewed?). Dashed lines mark the mean (the arithmetic average) and the median (the middle value). When mean and median sit far apart, the data is lopsided.

Use when: you want to understand how a single property is spread — typical values, range, and whether there are hidden sub-piles.

Box plot

A five-number summary in one shape: the box spans the middle half of the data (25th to 75th percentile), the thick line is the median, the "whiskers" reach the typical extremes, and stray dots beyond them are outliers. Compact way to compare several groups side by side.

Use when: you want to compare the spread and center of one property across several categories without plotting every point.

Violin plot

Like a box plot, but the width of the "violin" shows how many molecules sit at each value — wide where compounds bunch up, narrow where they thin out. A two-bulge violin (right) reveals two sub-populations a box plot would hide.

Use when: you suspect a property is not a single smooth pile and want to see the full distribution shape per group.

Correlation heatmap

A colored grid where each cell is the correlation between two properties — a number from −1 to +1 saying how strongly they rise and fall together (+1 = move identically, 0 = unrelated, −1 = opposite). Warm cells = strong positive, cool cells = strong negative. Scan it to find which properties are essentially duplicates.

Use when: you have many properties and want a one-glance map of which ones are linked before you build models or pick axes.

3-D scatter

Plots three properties at once (X, Y, Z) as floating points you can rotate, and can overlay a fitted regression plane — the flat surface that best passes through the cloud. Points above the plane sit higher than predicted; points below sit lower.

Use when: two axes aren't enough — you want to view three properties together. For the full statistics, use the dedicated 3D regression page.

Pair / grid comparison

A matrix of small panels: each off-diagonal tile is a scatter of one property against another, and the diagonal shows each property's own histogram. One screen lets you scan every pairwise relationship at once.

Use when: you're exploring a fresh dataset and want a fast overview of how a handful of properties relate to each other before zooming into one pair.

Encoding extra information

A flat scatter shows two properties (X and Y). But a molecule has many traits, and you can pack more of them into the same chart by mapping extra columns onto the look of each point. These are called encodings — each one is a separate dropdown in the Visualization controls.

Size‑by

Map a column to dot size — bigger dot = bigger value (e.g. larger dots for heavier compounds). Lets a third number ride along on the chart.

Shape‑by

Map a category column to point shape (circle, square, triangle…). Handy for separating classes — active vs inactive, scaffold A vs B — without relying on color alone.

Color‑by

Map a column to point color. For numbers it becomes a smooth gradient; for categories, distinct hues. The single most useful way to reveal a hidden third dimension.

Stack encodings carefully

You can combine encodings — e.g. X = weight, Y = activity, color = LogP, size = TPSA packs four properties into one chart. But more than three or four at once gets hard to read. If a plot feels cluttered, drop an encoding and lean on the pair grid instead.

Choosing a color scale (and reversing it)

When you color points by a number, the app maps low values to one end of a color scale (a fixed sequence of colors, sometimes called a colormap) and high values to the other. You can pick the scale from a dropdown and toggle Reverse to flip which end is "high." Why it matters: a well-chosen scale makes the pattern jump out, and reversing lets you put the color you mentally associate with "good" (or "danger") on the side you care about. It changes only the chart's appearance — never the underlying data.

Interacting with a chart

Because the charts are live (built with Plotly), every plot comes with a small toolbar and a set of mouse gestures. None of these change your data — they only change your view.

Hover for details

Float the cursor over any point to pop a tooltip with that molecule's exact X/Y values (and any encoded columns). No need to eyeball positions.

Zoom & pan

Drag a box to zoom into a crowded region, scroll to zoom in and out, and drag to pan around. Double-click to reset the view to the full chart.

Export as image

The camera button in the chart toolbar saves the current view as a PNG image — drop it straight into a slide, report, or notebook.

Toggle series

Click a label in the legend to hide or show that group — isolate one series, or strip away clutter to focus on the comparison you care about.

Click any point to see its structure

This is the feature chemists tend to love most. Every dot on a chart is a real molecule — so click a point and a 2-D structure pops up for that exact compound. The drawing is rendered right in your browser by SmilesDrawer.js (a lightweight JavaScript library that turns a SMILES string into a structure picture), so there's no waiting on a server and it works for any compound in your table.

The click-to-see-structure trick

Spotted a strange outlier in a box plot, or a point that defies the trendline? Click it. Instead of cross-referencing row numbers back to a spreadsheet, you instantly see the chemistry behind the dot — perfect for asking "what is special about that compound?" right where the question arises. It works on scatter points, 3-D points, and any chart where points are individual molecules. Try it on aspirin CC(=O)OC1=CC=CC=C1C(=O)O.

The quick distribution-analysis panel

Alongside the main chart, the Visualization mode offers a one-click distribution analysis panel: pick a column and it instantly summarizes how that property is spread — a histogram plus the headline numbers (mean, median, minimum, maximum, and spread). It's the fastest way to sanity-check a single column before you commit to a full chart.

A good first move

Before plotting two columns against each other, run the distribution panel on each one. If a property is wildly skewed or has a stray giant value, you'll spot it here — and decide whether to filter it or pick a different chart.

Where the data comes from

The Visualization mode charts whatever table is loaded — either properties you generated in Batch Processing, or your own uploaded CSV/XLSX file. Any numeric column can become an axis or an encoding; any text column can drive shape, color, or bar categories. If your file has structures but few numbers, calculate the 28 properties first so there's something to plot.
